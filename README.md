# Pradnya Jadhav — Portfolio

Personal portfolio website showcasing my work as a MERN Stack Developer. Built with Next.js 15, TypeScript, and Tailwind CSS.

## About

Final-year B.Tech Computer Science student (DBATU, Class of 2026) specializing in full-stack web development with React.js, Node.js, Express.js, and MongoDB.

## Tech Stack

- **Next.js 15** — App Router, React Server Components
- **React 19**
- **TypeScript**
- **Tailwind CSS** — utility-first styling with custom theme tokens

## Getting Started

1. Install dependencies:
```bash
   npm install
```

2. Start the development server:
```bash
   npm run dev
```

3. Open [http://localhost:4028](http://localhost:4028) in your browser.
